# پروژه رزومه شخصی

این پروژه شامل یک وب‌سایت رزومه شخصی است که با استفاده از FastAPI برای بخش بک‌اند و HTML/CSS/JavaScript برای بخش فرانت‌اند ساخته شده است.

## ساختار پروژه

- backend/: شامل کدهای پایتون برای API
- frontend/: شامل فایل‌های HTML، CSS و JavaScript برای نمایش در مرورگر

## نحوه راه‌اندازی

1. در پوشه backend، دستور uvicorn main:app --reload رو اجرا کنید.
2. در پوشه frontend، یک سرور ساده با دستور python -m http.server 5500 راه‌اندازی کنید.
3. پروژه رو در مرورگر با آدرس http://localhost:5500 مشاهده کنید
